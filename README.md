**Harbor (Airbnb Lite)**
Harbor — это микросервисная платформа для краткосрочной и среднесрочной аренды недвижимости. Название отражает суть проекта: поиск «тихой гавани» для путешественников и надежного инструмента для хостов.
Проект спроектирован с упором на масштабируемость, разделение ответственности (Bounded Contexts) и безопасность сделок

Проект построен на базе микросервисной архитектуры, где каждый сервис отвечает за свою бизнес-область в рамках методологии Domain-Driven Design (DDD)

Ключевые сервисы:
User & Identity Service: Управление профилями, ролями (Гость/Хост) и верификацией пользователей

Listing Service: Каталог объектов недвижимости, гео-поиск по координатам и управление контентом (фото, описания)

Booking & Availability Service: Сердце системы. Управляет календарем занятости, предотвращает двойные бронирования и рассчитывает стоимость

Communication Service: Внутренняя система чатов и уведомлений (Push/Email/SMS)

Review & Reputation Service: Двусторонняя система оценок и модерация спорных ситуаций

**Стек**

Backend: Go

Databases: PostgreSQL

Infrastructure: Docker, Kubernetes, Nginx (API Gateway)

Communication: Apache Kafka

**Текущий MVP**
Регистрация и верификация пользователей

Создание объявлений с привязкой к гео-координатам

Поиск жилья по датам и фильтрам

Механика бронирования с блокировкой календаря
